it will check .tf files and state file with actual infra 

if this is diff then terraform will make same config to actual infra with comparing to state file and .tf file
state is locked by terraform when it is performing action on it.

provisioners
=============
1. local-exec  --- > where terraform exectes( like we are ruuning now from windows say like from local exec)
2. remote-exec --->exectues insides the resources created by terraform.


if we want to run some scripts or take some actiom the we can use provisioners

provisioners will be executaed at the time of destroy time or creation  time but not time of update the resources

    provisioner "local-exec" {
    command = "echo ${self.public_ip} > inventory.ini"
  }
 
    provisioner "local-exec" {
    command = "echo  > inventory.ini"
    when = destroy
